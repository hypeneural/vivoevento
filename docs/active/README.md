# Active Feature Context

Use `docs/active/<feature>/` for live feature context only.

Recommended files:

- `STATUS.md`
- `DECISIONS.md`
- `VERIFY.md`

Template path:

- `docs/active/_template/`

Rules:

- keep the current state here while the feature is in flight
- move durable historical analysis to `docs/architecture/`
- keep executable implementation order in `docs/execution-plans/`
- store exact commands and validation evidence in `VERIFY.md`
