# Analytics Module
## Responsabilidade
Métricas e consolidados de uso por evento e plataforma.
## Entidades
- **AnalyticsEvent** — registro de eventos analíticos
## Rotas
| Método | Rota | Descrição |
|--------|------|-----------|
| GET | /api/v1/analytics/events/{id} | Métricas do evento |
| GET | /api/v1/analytics/platform | Visão geral da plataforma |
