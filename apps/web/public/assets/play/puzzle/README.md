# Puzzle Test Assets

Estrutura base para o modulo `puzzle` do Evento Vivo.

Pastas:

- `audio/`: sons curtos de pickup, hover, snap, erro e vitoria.
- `icons/`: icones simples para HUD e estados do jogo.
- `particles/`: sprites leves para emissor de particulas.
- `prompts/`: prompts de input para toque e arraste.

Estes arquivos sao placeholders de desenvolvimento. Na troca por assets finais:

- manter nomes e caminhos para evitar retrabalho de preload;
- priorizar formatos leves;
- evitar arquivos grandes para nao piorar o primeiro carregamento em celular simples.
